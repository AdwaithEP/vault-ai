# Vault AI

An AI-powered cybersecurity platform that combines:

- 🔐 Password Vault
- 🤖 AI Password Suggestions
- 🌐 URL Scanner (VirusTotal API)
- 📡 Packet Tracer


## Technology Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Django

### Database
- Supabase (PostgreSQL)

### APIs & Services
- VirusTotal API
- AI Password Assistant

### Tools
- Git
- GitHub
- VS Code
- 
## Features

- Secure password management
- AI-generated strong passwords
- URL reputation scanning
- Network packet analysis

## Installation

1. Clone the repository
2. Install requirements
3. Run the Django server


## Future Improvements

- Multi-factor authentication
- Browser extension
- Cloud sync
